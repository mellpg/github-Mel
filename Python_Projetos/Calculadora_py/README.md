# Calculadora Python (Console)

## Descrição
Este projeto é uma calculadora de linha de comando em Python

## Estrutura do Código
-'calculadora.py' -> Contém a classe 'Calculadora'
-'main.py' -> script principal que interage com o usuário

## Funcionamento
- O usuário escolhe a operação desejada a partir de um menu.
- Para operações como **porcentagem** e **resto**, o programa solicita exatamente dois números.
- Para as demais operações, o usuário pode inserir múltiplos números separados por espaço.
- O programa trata erros de entrada (não numérico) e divisão por zero.


  ## Tecnologias utilizadas
  - python3
  - Programação orientada a objetos (OOP)
  - Estruturas de repetição: Loops.
  - Variáveis, atribuições e Strings.
  - Estruturas de condição
  - Contadores/Acumuladores
  - Interação com o usuário e programa principal.
